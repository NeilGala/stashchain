# ⛓️ StashChain

A decentralized supply chain management system built on Ethereum.

## 🔗 Live Demo
https://stashchain.vercel.app/

## 📄 Smart Contract
- Network: Ethereum Sepolia Testnet
- Address: 0xCE8ce27c811a7acc6D64543112af2f64EfbE5460
- [View on Etherscan](https://sepolia.etherscan.io/address/0xCE8ce27c811a7acc6D64543112af2f64EfbE5460)

## 🛠️ Tech Stack
- **Frontend:** Next.js 16 + Tailwind CSS v4
- **Smart Contracts:** Solidity 0.8.20 + Hardhat 2
- **Web3:** Ethers.js v6
- **Storage:** IPFS via Pinata
- **Blockchain:** Ethereum Sepolia Testnet
- **Deployment:** Vercel

## ✨ Features
- 🔐 MetaMask wallet authentication
- 📦 Product registration with IPFS metadata
- 📋 Supply chain event logging (Manufactured, Shipped, In Transit, Delivered)
- 🤝 Ownership transfer between wallets
- 🔍 Full product history timeline
- ✅ On-chain authenticity verification

## 🚀 Local Development

### Prerequisites
- Node.js v20+
- MetaMask browser extension
- Sepolia test ETH

# Install blockchain dependencies
cd blockchain
npm install

# Install frontend dependencies
cd ../frontend
npm install

# Start frontend
npm run dev
